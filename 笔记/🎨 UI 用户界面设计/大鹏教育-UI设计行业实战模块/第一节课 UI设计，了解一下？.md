### 01 认识 UI 设计

UI = User Interface 即用户界面，包含移动APP，网页，智能穿戴设备界面等。

汽车 仪表盘、点餐系统、手机、电脑、iPad...

UI设计主要指界面的样式，美观程度。

Web UI，游戏 UI，工业 UI

App 设计占到很大一部分

UI 只考虑做好美观的页面就够了吗？
只美观 - 艺术品；
只好用 - 工具；

柏拉图说，美是让人产生愉快心情的感受
